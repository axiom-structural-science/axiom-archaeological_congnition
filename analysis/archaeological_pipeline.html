2. Structural Analysis

Calculate STI score for archaeological site cognitive complexity

function calculateArchaeologicalSTI(concepts, siteBP) { // Era-appropriate scoring const expectedConcepts = getConceptsForEra(siteBP); const presentConcepts = concepts.filter(c => expectedConcepts.includes(c)); return presentConcepts.length / expectedConcepts.length; }

3. Temporal Positioning

Assess if site's cognitive development matches its era

function assessCognitiveEra(concepts, siteBP) { const latestConcept = Math.max(...concepts.map(c => getConceptEra(c))); const eraAppropriateness = 1 - Math.abs(latestConcept - siteBP) / 100000; return Math.max(0, Math.min(1, eraAppropriateness)); }

function archaeologicalAXIOMAnalysis(siteData) { const concepts = mapArtifactsToConcepts(siteData.artifacts); const sti = calculateArchaeologicalSTI(concepts, siteData.bp); const eraFit = assessCognitiveEra(concepts, siteData.bp); const evidenceQuality = evaluateEvidence(siteData); return { concepts: concepts, sti_score: sti, era_appropriateness: eraFit, evidence_quality: evidenceQuality, interpretation_confidence: (sti + eraFit + evidenceQuality) / 3 }; }
